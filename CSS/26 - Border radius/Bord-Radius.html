Border radius en CSS

La propiedad de CSS Border-radius es un atributo que redondea las esquinas de un elemento y ademas es un atributo "Shorthand" lo que indica que un valor universal en este atributo engloba todos los valores posibles de aplicacion tales como:

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius

Al hablar con la palabra radius si se hace referencia a un radio de un circulo ya que al indicar un border-radius de 24px se estar√° dibujando un circulo en la esquina del elemento con un radio de 24px

  • En HTML:

        <div class="div-1">
        Ejemplo 1
    </div>

        .div-1{
