Padding en CSS

La propiedad CSS padding establece el espacio de relleno requerido por todos los lados de un elemento. El Ã¡rea de padding es el espacio entre el contenido del elemento y su borde (border). No se permiten valores negativos.

tambien puede constar de cuatro valores especificables, ademas es una propiedad "Shorthand" lo que quierer decir que es capaz de aceptar solo uno de los cuatro valores especificables (es compatible con 1, 2, 3 o 4 valores)

los valores digitados en css corresponden a:

4 valores padding: top right bottom left

3 valores padding: top left / right bottom

2 valores padding: top / bottom left / right

1 valor padding: top / bottom / left / right

Ejemplo:

  1. En HTML:

                    
    <div class="block-1">Prueba de padding </div>

  2. En CSS:

                    
    .block-1{

        width: 100px;
        height: 100px;
        background-color: salmon;
        margin-top: 20px;

        padding: 100px 10px 40px;
        
    }
