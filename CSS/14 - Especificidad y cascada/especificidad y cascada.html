  • Especificidad:

    Hace referencia a como de especifico es un selector para saber que estilo aplicar, si se tienen estas formas de dar estilo en un documento CSS:

    1. En HTML: 
      
        <h1 id = "title" class = "title"> Fundamentos de CSS </h1>
    2. EN CSS: 
      
        h1{
            background-color: red;
            border-radius: 4px;
        }
        
        .title{
            background-color: aqua;
            border-radius: 4px;
        }
        
        #title{
            background-color: yellow;
            border-radius: 4px;
        }
        
        h1.title {
            background-color: gray;
            border-radius: 4px;
        }
        
        h1 #title{
            background-color: darkorchid;
            border-radius: 4px;
        }
        
        h1.title#title{
            background-color: saddlebrown;
            border-radius: 4px;
        }

El estilo que se aplicara sera el que tiene el selector " h1.title#title " y esto no es debido al efecto cascada

Esto se debe a que cada uno de los elementos de seleccion (ya sea h1 o .title o #title etc) tiene un valor numerico y el navegador hace un calculo y suma los valores, siendo la estructura CSS con el valor mas grande el que tinene mas prioridad

los valores de los elementos son:

Etiquetas y seudoelementos 001

Clases, atributos y pseudoclases 010

Id's 100

Estilos en linea 1000

Agregar un !important a la estructura CSS Gana a todos ya que rompe la especificidad y toma toda la prioridad (No se usa nunca)

Nota: Lo correcto es siempre usar clases.

Es decir que si un selector intentar dar estilos con un ".title" tiene un valor de especificidad de 010 y si en otra estructura se intenta dar estilo con un "h1.title#title" tiene un valor de 111 asi que este toma la prioridad de especificidad ya que tiene los valores de etiquetas, clases y id's juntos

Nota: El efecto "Cascada" afecta igual en la especificidad pero en las estructuras que tengan la misma especificidad, en ese caso los estilos se aplicaran sobre escribiendose unos a otros.
