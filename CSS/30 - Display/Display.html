Display

La propiedad displa se usa entre otras cosas para poder "Cambiar" la naturalidad de un elemento de linea o de bloque haciendo que un elemento de linea se pueda convertir en un elemento de bloque y bisceversa, en realidad no se cambia de raiz su naturalidad pero se le asignan las caracteristicas que tiene el elemento que se escoja ya sea de linea o de bloque

Consta de los siguientes valores:

  • none -> Hace que el elemento no se muestre, pero sigue cargándose.
  • block -> Hace que el elemento sea de bloque.
  • inline -> Hace que el elemento sea de linea.
  • inline-block -> Hace que el elemento sea de linea pero admite medidas y márgenes verticales.

Ejemplo (none):

  • En HTML:

        <p class="text-1"> Lorem ipsum dolor sit amet </p>

  • En CSS:

        .text-1{
        display: none;
    }

  • Resultado:

    Lorem ipsum dolor sit amet


    No se muestra ninguna Resultado por que esta propiedad da la orden de no dibujar el contenido en la pantalla pero si se esta cargando en el navegador por lo que si esta consuminedno recursos

Ejemplo (block):

  • En HTML:

        <a href="#" class="link">Enlace </a>

  • En CSS:

        .text-1{
        display: block;
        padding-bottom: 24px;
    }

  • Resultado:

    Enlace


    El tamaño del elemento que inicialmente era de linea ahora es el tamaño de un elemento de bloque ya que se le otergaron los atributos de un elemento de bloque, y por lo tanto su tamaño también puede cambiar

Ejemplo (inline):

  • En HTML:

        <p class="text">Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>

  • En CSS:

        .text-2{
        display: inline;
        margin-bottom: 320px;
    }

  • Resultado:

    Lorem ipsum dolor sit amet consectetur adipisicing elit.


    El elemento que inicialmente era de bloque ahora adopto las caracteristicas de un elemento de linea es decir que ahora no se puede modificar el tamaño del elemento

Ejemplo (inline-block):

  • En HTML:

        <a href="#" class="link">Enlace</a>

  • En CSS:

        .link-2{
        display: inline-block;
        padding-bottom: 38px;
        padding-right: 124px;
    }

  • Resultado:

    Enlace


    El elemento que inicialmente era de linea ahora adopto las caracteristicas de un elemento de bloque es decir que ahora se puede modificar el tamaño del elemento con atributos como
