Border en CSS

La propiedad border se usa para modificar los bordes de un elemento ya que por defecto los elementos ya tienen borde pero no tiene valor, es un elemetno "Shorthand" y engloba los atributos border-width, border-style y border- color.

dentor de las tres propiedades que puede agrupar estan:

border: border-width border-style border-color

y de estos tres atributos los tres son "shorthands" ya que engloban el atributo de cada uno de los lados en todos los casos es decir que al especificar un valor para border-width se esta especificando el valor de los cuatro lados del elemento lo que tambien hace posible que se puedan especificar datos independientes para todos los lados ya que son independientes.



Es posible hacer esto con todos los atributos tales como: 

        
    border-top-width: ;
    border-right-width: ;
    border-bottom-width: ;
    border-left-width: ;

    border-top-style: ;
    border-right-style: ;
    border-bottom-style: ;
    border-right-style: ;

    border-top-color: ;
    border-left-color: ;
    border-bottom-color: ;
    border-right-color: ;

Listado de valores del atributo style:

El atributo style puede tener varios valores es decir que una linea puede tener diferentes estilos, los estilos que puede tener un borde son:

  1. None:

    Este atributo representa un margen nulo es decir que aunque se le asigne un valor este no asignara ningun tipo de margen.

    Ejemplo:

  2. Hidden:

    Este atributo representa un margen casi nulo pero tiene diferencias en cuanto al margen none a la hora de ser usado en margenes para imagenes.

    Ejemplo:

  3. Dotted:

    Este atributo representa un margen hecho de puntos y el atributo width representa el diametro de cada uno de estos circulos.

    Ejemplo:

  4. Dashed:

    Este atributo representa un margen hecho de rectangulos y el atributo width representa la anchura y altura de cada uno de estos rectangulos.

    Ejemplo:

  5. Solid:

    Este atributo representa un margen completo y solido es decir que el width solo represntara el ancho del margen.

    Ejemplo:

  6. Double:

    Este atributo representa un margen de doble linea dejando un espacio sin color en el medio y el atributo width es dividido en tres partes iguales, una para el ancho del primer margen, otra para el ancho de la parte que ira sin margen y otra parte para el margen restante.

    Ejemplo:

  7. Groove:

    Este atributo representa un margen en forma de sombra 3D en la que cada uno de los lados se divide en dos extremos y uno estara mas sombreado que otro dando la ilusion de ser un elemento 3D.

    Ejemplo:

  8. Ridge:

    Este atributo tambien representa un margen en forma de sombra 3D igual que el anterior pero los lados en los que se ve la sombra estan invertidos, cada uno de los lados se divide en dos extremos y uno estara mas sombreado que otro dando la ilusion de ser un elemento 3D.

    Ejemplo:

  9. Inset:

    Este atributo tambien representa un margen en forma de sombra 3D pero no divide el borde en dos partes, en su lugar pinta dos de los lados del elemento de un color mas oscuro dando la ilusion de ser un elemento 3D.

    Ejemplo:

  10. Outset:

    Este atributo tambien representa un margen en forma de sombra 3D igual que el anterior pero con los colores de la sombra invertidos dando la ilusion de ser un elemento 3D.
