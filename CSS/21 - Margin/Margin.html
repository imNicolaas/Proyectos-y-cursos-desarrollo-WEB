Margin en CSS

¿Que es el margin?

La propiedad CSS margin establece el margen para los cuatro lados. Es una abreviación para evitar tener que establecer cada lado por separado con las otras propiedades de margen: margin-top, margin-right, margin-bottom y margin-left.

Width y Height en elementos de bloque:

Para los elementos de bloque el margin tiene un margen de actuacion completo, es decir que todas las formas de decalran un margen en todos los lados tendra un efecto

Los cuatro valores que se pueden establecer en un atributo margin actuan de la siguiente forma:

  1. forma #1:

    • CSS:

      
    .block-1{
        background-color: steelblue;
        width: 100px;
        height: 100px;
        
        margin: 50px;
    }

      Cuando se pone solo un valor en el atributo margin este se establece como el valor de todos los cuatro margenes

  2. Forma #2:

    • CSS:

      
    .block-2{
        background-color: steelblue;
        width: 100px;
        height: 100px;
        
        margin: 100px 100px;
    }

      Cuando se ponen dos valores en el atributo margin el primero se establecera como el valor del margen superior e inferior y el segundo valor se establcera como el valor del margen derecho e izquierdo

    • Resultado:

      Elemento
      de bloque

  3. forma #3:

    • CSS:

      
    .block-3{
        background-color: steelblue;
        width: 100px;
        height: 100px;
        
        margin: 150px 160px 50px 40px ;
    }

      Cuando se ponen cuatro valores cada uno hace referencia a un valor de los cuatro margenes posibles y el orden en que estos valores toman lugar es siguiendo el sentido del reloj es decir que el primer valor tomara valor del margen de arriba, el segundo valor tomara el valor del margen derecho, el tercer valor tomara el valor del margen inferior y el cuarto valor tomara el valor del lado izquierdo.

    • Resultado:

      Elemento
      de bloque

  4. forma #4:

    • CSS:

      
    .block-auto{
        background-color: steelblue;
        width: 100px;
        height: 100px;
        
        margin: 150px 20px auto auto;
    }

      Cuadno se hace uso del atributo auto para rellernar uno de los valores del margin (independientemente del los que se esten usando), este margen se calculara automaticamente por ejemplo si hay solo un valor y se pone auto, le dara la misma cantidad equitativamente de la cantidad de espacio disponible a cada una de las direcciones del margen

      y si se tienen dos valores se comportara de la misma forma en el que se otorgue el atributo auto

      De la misma forma se comportara si se usa el atributo auto en uno de los cuatro valores a especificar, si se especifican 150px en el margen superior de los 200px disponibles, al poner el atributo auto en el margen inferior este le otorgara los 50px que faltan a este margen

Margin en elementos de linea:

A diferencia de los elementos de bloque, los elementos de linea no tienen medidas, o no tinene medidas modificables es decir que su ancho y alto van a estar definidos por el tamaño del contenido

Si intentamos modificar el width y el height de un elemneto de linea, estas no tendran efecto sobre de el:

  • HTML: 
    
    <a href="#" class = "inline"> Elemento de linea </a>
  • CSS: 
    
    .inline{
        background-color: steelblue;
        width: 200px;
        height: 100px;

        margin: 150px 20px;
    }
  • Resultado:

    Elemento de linea
