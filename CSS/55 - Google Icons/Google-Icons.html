Google Icons en CSS

Google icons es la plataforma de google que nos proporciona iconos a los desarrolladores.

Existen varias librerias de iconos dentro de google icons tales como Outlined, Rounded y Sharp

si se van a usar una gran cantidad de iconos en la web de esa libreria si vale la pena utilizar

Pero si solo se van a usar unos pocos es mejor descargarlos en formato svg y utilizarlos como imagenes

Ejemplo:

fingerprint print_disabled

para darle estilos a un icono hay que hacerlo como se el daria estilos a cualquier otra cosa en css ya sea con clases o id, pero dandole las instrucciones como si fueran fuentes ya que en el fonde eso es lo que son.

electric_scooter
