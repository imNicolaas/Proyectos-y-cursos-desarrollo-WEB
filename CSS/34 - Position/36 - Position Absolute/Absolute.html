Position Absolute en CSS

Al hacer uso de esta propiedad en CSS en un elemento este pierde sus medidas y espacio reservado, Si lo movemos usará el elemento contenedor como referencia (SIEMPRE Y CUANDO SU CONTENEDOR ESTE POSICIONADO, CON CUALQUIERA DE LOS VALORES DE POSITION, DE LO CONTRARIO NO VA A FUNCIONAR.), y si no tiene un elemento contenedor usará elemento HTML como referencia

Ejemplo (perdida de medidas):

  • Antes:

    Box 1
    Box 2

  • En HTML

        <div class="box-1-antes"> Box 1 </div>
    <div class="box-2-antes"> Box 2 </div>

  • En CSS:

    .box-2{
    background-color: lightgreen;
    position: absolute;
}

  • Resultado (Después):

    Box 1
    Box 2


    Nota: El elemento al que se aplica la propiedad Position absolute pierde sus medidas

Si un elemento no tiene contenedor usara el html como contenedor, si este elemento estea dentro del body este sera su contenedor, tenga el tamaño que tenga.
