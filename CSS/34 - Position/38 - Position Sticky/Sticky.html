Position Sticky CSS

Es una mezcla de position Relative y position Fixed, Con este tipo de posicionamiento los valores Top, left, bottom, y right no funcionan para mover el elemento, sino para indicarle en que punto en que punto pasara a tener un comportamiento de posicionamiento similar a fixed, hasta llegar a ese punto se comportara como si no tubiera posicionamiento (es decir antes de que se cumpla la condicion que estamos especifcancon de Top, left, bottom o right el elemento si sera suceptible a el scroll), Pero una vez que se cumpla la condicion que estamos estipulando el elemento si se comportara similar a fixed

Caracteristicas:

Esta propiedad evalua las medidas del contenedor en el que esta es decir que si el elemento no tiene contenedor no tiene medidas, esta propiedad no funcionara

y si el contenedor del elemento si tiene medidas pero se le aplica la propiedad position Sticky a el elemento en si, es decir a lo que esta por dentro se cumplira pero dentro de su contenedor (ademas de las condiciones de Top, left, bottom o right), de esta forma.

  • En HTML:

        <div class="prueba-1">
        <nav class="nav-prueba-1">Prueba 1</nav>
    </div>

  • En CSS:

    .prueba-1{
    height: 240px;
}
                
.nav-prueba-1{
    position: sticky;
    top: 75px;
}

  • Resultado:

Prueba 1

El elemento hijo esta cupliendo la condicion de quedarse sticky cuando el top sea 75px pero solo mientras la altura de su elemento padre se lo permita, cuando no lo haga tendra que perder la propiedad Stiky ya que su elemento padre no tiene altura y ademas no se puede mover con el.

En el caso de position fixed toma como referencia al elemento padre HTML o body siempre, en el caso de sticky toma como referecia las medidas del elemento padre.

En el siguiente ejemplo se tienen etiquetas article en las que solo hay un h2 con el nombre de la seccion y una etiquta p con el contenido del parrafo. Al aplicar la propiedad Sticky al h2 va a tener poder de actuacion mientras su elemento padre (la etiqueta article) tenga una altura en la que se pueda representar su "fijacion" de la propiedad Sticky cuando este article se quede sin altura se dejara de representar la fijacion y perdera su posicionamiento en la pantalla pasando a ser un elemento normal, y el proximo h2 cumplira con las condiciones establecidas por lo tanto se le aplicara la propiedad Sticky en pantalla, y asi sucesivamente. IMPORTANTE: Cuando el elemento padre al que se le esta aplicando la propiedad Sticky tiene la propiedad Overflow declarada con cualquier valor que no sea "visible" la propiedad Sticky no funcionara, es decir hay que tener en cuenta si el contenido tiene el alto sufuciente para generar un Scroll
