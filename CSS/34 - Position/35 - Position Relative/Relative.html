Position Relative en CSS

El elemento al que se le aplique esta propiedad o cualquiera de sus atributos mantendra su posicion y medidas en el flujo de renderizado y mantandra su espacion reservado, si lo movemos el ancla o referencia del movimiento es la posicion en el HTML es decir el punto (0, 0) que es la esquina superior izquierda.

A la hacer uso de la propiedad position: relative; se habilitan valores como:

  1. left
  2. right
  3. top
  4. bottom

y todos puden tomar valores negativos o positivos

Ejemplo

  • En HTML

    <div class="box-1"></div>

  • en CSS

    .box-1{
    width: 160px;
    height: 160px;

    position: relative;
    left: 132px;
    top: 32px;
}

  • Resultado:

    box 1
