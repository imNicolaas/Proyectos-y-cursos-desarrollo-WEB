height % y Auto en CSS

Height con Porcentaje %:

Cuando estamos dentro de un contenedor, este tiene que tener un alto declarado, si no no podrá calcular el porcentaje. Si se Le aplica una medida con porcentaje se sustituirá automaticamente por auto.

Cuando se sustituye por auto automaticamente el alto se va a calculara en base al contendio del elemento, es decir que tomara el alto necesario para mostrar el contendido que tenga dentro, si no hay contendo el height esta en auto, directamente no se mostrará.

Height % con Position absolute y fixed:

Cuando se especifica position absolute o fixed en un elemento y al mismo tiempo un height en porcentaje, automaticamente el height se calcula con respecto al viewport, es decir al espacio que hay para visualizar y dibujar cosas, pero si se especifica position Relative o Sticky las cosas vuelven a la normalidad.
