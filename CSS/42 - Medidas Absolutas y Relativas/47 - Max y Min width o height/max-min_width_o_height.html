max-width/height y min-width/height en CSS

Limitadores de ancho y alto en css

Cuando se establece una medidad ya sea de alto o ancho que sea relativa aveces se quiere que esta cresca o se enconja hasta cierto punto y que nunca supere o descienda mas de esa medida, es en estos casos donde se usa el max max-width/height y min-width/height

  • min-width: Indica el ancho minimo que puede alcanzar el elemento.
  • max-width: Indica el ancho máximo que puede alcanzar el elemento.
  • min-height: Indica el alto minimo que puede alcanzar el elemento.
  • max-height: Indica el alto máximo que puede alcanzar el elemento.

  • min width y mas width:

Si se declara un max width de 900px en este caso y como el width es relativo va a ir creciendo hasta un maximo de 900px, y con un min width de 510px ira decreciendo hasta que mida como minimo 510px, de ahi no bajará mas y si la ventana se hace mas pequeña que eso aparecerá una barra de scroll para que se pueda serguir viendo el elemento con un min width declarado

  • min width min height:

    • En HTML:

      <div class="box-2"> Box 2</div>

    • En CSS:

      .box-2{
    width: 100%;
    height: 20vw;
    
    max-height: 200px;
    min-height: 150px;

}

Truco:

Cuando se quiere dejar un elemento siempre abajo de la pagina uno pensaria en usar la propiedad position: absolute; y un bottom: 0; pero si se añade mas contenido a la pagina entonces esto no funcionaria, despues si se intenta lo mismo pero aplicando position: relative; AL BODY, se estará mas cerca de la solucion pero en algunos casos esto tampoco funcionará bien ya que el body no mide lo que se espera, asi que ademas de todo esto se le debera dar un min-height: 100vh; y de esta forma se le esta diciendo al body que como minimo mida el 100% del viewport.
