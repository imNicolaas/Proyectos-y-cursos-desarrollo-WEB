Width y Heigth en CSS

¿Que es el width y el height?

  • Width:

    La propiedad CSS width especifica la anchura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento.

    Las propiedades min-width y max-width sobreescriben el valor de width.

  • Height:

    La propiedad CSS height especifica la altura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento.

    Las propiedades min-height y max-height sobreescriben el valor de height.

Width y Height en elementos de bloque:

Para los elementos de bloque ek ancho y el alto (Width y height) si tienen potestad sobre el elemento en si, es decir que de ser definidos sus valores si van a determinar el ancho y alto del objeto

De esta forma:

  • HTML: 
    
    <div class = "block"> Elemento de bloque </div>
  • CSS: 
    
    .block{
        background-color: steelblue;
        width: 200px;
        height: 100px;
    }
  • Resultado:
Elemento de bloque

Width y Height en elementos de linea:

A diferencia de los elementos de bloque, los elementos de linea no tienen medidas, o no tinene medidas modificables es decir que su ancho y alto van a estar definidos por el tamaño del contenido

Si intentamos modificar el width y el height de un elemneto de linea, estas no tendran efecto sobre de el:

  • HTML: 
    
    <a href="#" class = "inline">Elemento de linea</a>
  • CSS: 
    
    .inline{
        background-color: steelblue;
        width: 200px;
        height: 100px;
    }
  • Resultado:

    Elemento de linea
