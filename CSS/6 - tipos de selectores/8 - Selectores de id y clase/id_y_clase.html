Selectores de id y clase en CSS

Selector por medio de id:

El id es un atributo por el cual se puede identificara una etiqueta en particular de las demas y se diferenciara de lo demas por que el id es unico para cada elemento, y a la hora de dar estilo a este objeto por medio de CSS se llamara a este id con un # pero no es muy recomendable usar id para dar estilos por que los "id's" son identificadores y esto quiere decir que solo puede haber uno, y uno de los principios de CSS es que los estilos sean reutilizables lo cual no es compatible con dar estilo a solo un elemento

De esta forma:

  • En HTML: 
    
    <h1 id="titulo-1">Selectores de id y clase en CSS
  • En CSS: 
    
    #titulo-1{
        display: flex;
        justify-content: center;
        background-color: lightslategray;
        border-radius: 5px;
    }

Selector por medio de class:

El class es un atributo que permite separar el elemento para acceder al el por medio de (en este caso) CSS para dar estilos y se accede a ellos en CSS por medio de un punto ( . ) y despues el nombre de la class

De esta forma:

  • En HTML: 
    
    <h1 class="titulo-1">Selectores de id y clase en CSS
  • En CSS: 
    
    .titulo-1{
        display: flex;
        justify-content: center;
        background-color: lightslategray;
        border-radius: 5px;
    }

No hay un limite de clases para un elemento o etiqueta, podemos poner las que queramos, as√≠: 


    <h1 class = "titulo-1 titulo-2 titulo-3 titulo-4" >Selectores de id y clase en CSS

Nota: las clases y id's son "Case sensitive" es decir que su alguna de las clases o id's es escrita con alguna de sus letras en mayuscula o minuscula cuando en su nombre esta de otra forma, el lenguaje CSS no entendera que se habla de el elemento en si.
