Selectores de atributos en CSS

Selector por uso de [ ]:

Se utilizan llavez tipo [ ] y dentro se pone el atributo que queremos seleccionar.

  • En HTML: 
    
    <li> ‎viernes, ‎18 ‎de ‎junio ‎de ‎2021 - ‏‎6:41:17 pm: 
        <a href="/HTML y CSS/HTML/Etiqueta_audio.html" >
        Etiqueta_audio.html </a> </li>
  • En CSS: 
    
    [href]{
        background-color: lightgray;
        border-radius: 4px;
    }
  • Resultado:
  1. ‎viernes, ‎18 ‎de ‎junio ‎de ‎2021 - ‏‎6:41:17 pm: Etiqueta_audio.html
  2. ‎viernes, ‎18 ‎de ‎junio ‎de ‎2021 - ‏‎7:49:19 pm: Etiqueta_video.html
  3. ‎domingo, ‎20 ‎de ‎junio ‎de ‎2021 - ‏‎5:55:22 pm: Etiqueta_Iframe.html
  4. ‎jueves, ‎17 ‎de ‎junio ‎de ‎2021 - 7:43:35 pm: Etiqueta_Picture.html

Selector por medio de [atributo] y su valor:

O tambien se puede poner el nombre del atributo y su valor para cambiar solo una propiedad especifica.

De esta forma:

  • En HTML: 
    
    <li>domingo, ‎27 ‎de ‎junio ‎de ‎2021 ‏‎12:30:06 pm: 
        <a href="/HTML y CSS/HTML/Open_Graph_Protocol.html"> 
        Etiqueta_Iframe.html </a> </li>
  • En CSS: 
    
    [href="/HTML y CSS/HTML/Open_Graph_Protocol.html"]{
        background-color: salmon;
        font-size: 21;
    }
  • Resultado:
  1. ‎viernes, ‎18 ‎de ‎junio ‎de ‎2021 - ‏‎6:41:17 pm: Etiqueta_audio.html
  2. ‎viernes, ‎18 ‎de ‎junio ‎de ‎2021 - ‏‎7:49:19 pm: Etiqueta_video.html
  3. ‎domingo, ‎27 ‎de ‎junio ‎de ‎2021 ‏‎12:30:06 pm: Open_Graph_Protocol.html
  4. ‎jueves, ‎17 ‎de ‎junio ‎de ‎2021 - 7:43:35 pm: Etiqueta_Picture.html

Selector por medio de [ atributo ] y coincidencia de nombre al inicio:

Para seleccionar elementos con la sintaxis de [ ] y una coincidencia de nombre se usa el acento circunflejo (^) despues del nombre de el atributo para indicar que queremos seleccionar todos los elementos cuyo nombre empiece por lo indicado en el valor dentro de las comillas (href^ = "texto a coincidir en el inicio ")

De esta forma:

  • En HTML: 
    
    <a href = "color-verde-oscuro"> Color verde oscuro </a>

    <a href = "color-de-relleno-cabecera"> Color de relleno para cabecera </a>
  • En CSS: 
    
    [href^ ="color"]{
        background-color: lightgreen;
        font-family: monospace, sans-serif;
    }
  • Resultado:
Color verde oscuro
Color de relleno para cabecera

Selector por medio de [ atributo ] y coincidencia de nombre al en cualquier lugar del enunciado:

Para hacer una seleccion de los elementos con una coincidencia en el nombre en cualquier parte de el nombre del "valor del atributo" se usa un * como acompañante del atributo que se vaya a usar para seleccionar los elementos

de esta forma:

  • En HTML: 
    
    <a href = "color-azul-cabeza"> Color azul cabeza </a>

    <a href = "cabeza-de-relleno-articulo"> Cabeza de relleno articulo </a>
  • En CSS: 
    
    [href* ="cabeza"]{
        background-color: red; 
    }
  • Resultado:
Color azul cabeza
Cabeza de relleno articulo

Selector por medio de [ atributo ] y coincidencia de nombre al final del enunciado:

Para hacer una seleccion de los elementos con una coincidencia en el nombre en el final del nombre del "valor del atributo" se usa un signo$ como acompañante del atributo que se vaya a usar para seleccionar los elementos

de esta forma:

  • En HTML: 
    
    <a href = "motor-de-plantilla"> motor de plantilla </a>

    <a href = "color-de-relleno-plantilla"> color de relleno plantilla </a>
  • En CSS: 
    
    [href$ ="plantilla"]{
        background-color: steelblue;
        border-radius: 4px;
    }
  • Resultado:
motor de plantilla
color de relleno plantilla

Selector por medio de [ atributo ] y el simbolo " | " :

Este tipo de seleccion se usa generalmente para los elementos que tienen atributos donde se especifica el idioma en el que esta escrito, puede funcionar solo con la coincidencia de las siglas del idioma en si, y haciendo uso del simbolo " | " se le puede agregar un guion el valor del atributo lang o idioma y esta seleccion seguira funcionando.

de esta forma:

  • En HTML: 
    
    <p lang="es-CO">
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Nobis quasi 
            veniam, velit perferendis dolor ad hic, soluta doloribus autem ex tenetur 
            voluptates qui mollitia eveniet adipisci, quod et asperiores eum?
    </p>
  • En CSS: 
    
    [href|="es"]{
        background-color: burlywood;
    }
  • Resultado:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsum nihil praesentium iusto sed quaerat nostrum magni, neque optio dignissimos ex, consequatur incidunt eaque hic harum autem. Vitae quae hic tempore! Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci, earum nesciunt ullam unde fuga praesentium facilis itaque eius, expedita nam molestias dolore molestiae impedit excepturi. Sit illo tempore ipsa sapiente?

Nota: Todas estas formas de seleccionar elementos son suceptibles al efecto "Cascada".
