Border radius en CSS

En este caso el border radius eliptico funciona de manera similar al border radius normal pero en vez de dibujar un circulo con un radio especifico esta propiedad dibuja una elipse a la que se le deben asignar dos valores (el radio en X y el radio en Y), de esta fomra:

  • border-top-left-radius: (Valor 1) (Valor 2)
  • border-top-right-radius: (Valor 1) (Valor 2)
  • border-bottom-right-radius: (Valor 1) (Valor 2)
  • border-bottom-left-radius: (Valor 1) (Valor 2)

Para hacer uso del Shorthand es decir especifcar los valores de todos los lados con una sola instruccion la sintexsis es algo diferente ya que se debara indicar el valor del radio en X seguido de el simbolo "/ " y el valor del radio en Y

Ejemplo:

  • En HTML:

        <div class="div-1">
        Ejemplo 1
    </div>

  • En CSS:

        .div-1{
        border-radius: 24px / 36px;
    }

  • Resultado:

    Ejemplo 1
