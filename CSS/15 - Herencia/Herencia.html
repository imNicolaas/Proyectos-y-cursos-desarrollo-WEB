Herencia en CSS

Es la capacidad que tienen algunos elementos de heredar algunas propiedades de los elementos en los cuales estan contenidos (ya sean padres, abuelos, etc).

Por ejemplo si se tiene esta estructura de html y css:

  • HTML: 
    
    <h2> Esto es la herencia en <span> CSS </span> </h2>

    <span> CSS </span>
  • CSS: 
    
    h2{
        color: steelblue;
        font-size: 26px;
    }
  • El resultado que obtenemos es este:

Esto es la herencia en CSS

CSS

Como se puede apreciar el texto " CSS " que esta dentro de la etiqueta h2, heredó los estilos que se le estan apliacando a la etiqueta h1, y se hace de esta forma para tener una mayor armonia dentro de los estilos CSS ya que de lo contrario aplicar estilos a los elementos seria muy caótico. Ademas la mayoria de los estilos que CSS decide hacer que se puedan heredar por otros elementos tienen mucha coherencia en si.

Forzar la herencia a un elemento:

Pero no todos los elementos son seceptibles a la herecia, por ejemplo los elementos <a href=""> </a> no so suceptibles a una herencia como la anterior, para darle estilo a un href es necesario darselo a su clase propia o a toda la etiqueta en si (es decir directamente)

De esta forma:

  • HTML: 
    
    <p>
        Lorem ipsum dolor sit amet consectetur, adipisicing elit. Distinctio porro
            quos fugit eum corporis veritatis illo facilis, natus, reprehenderit <a
            href="#">     Mas sobre este articulo</a> necessitatibus autem, enim doloribus
            corrupti fuga modi impedit aspernatur commodi dicta.
    </p>
  • CSS: 
    
    p{
        color: deeppink;
    }
  • El resultado que obtenemos es este:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Distinctio porro quos fugit eum corporis veritatis illo facilis, natus, reprehenderit Mas sobre este articulo necessitatibus autem, enim doloribus corrupti fuga modi impedit aspernatur commodi dicta.

El estilo no se hereda al href pero hay una forma de "Forzar" a la herencia a funcionar con este elemento y es haciendo uso de atributo inherit el cual poniendolo dentro de el estilo CSS de el elemento que no fue afectado por la herencia, ahra tendra un color heredado de su "padre" o su contenedor

De esta forma:

  • HTML: 
    
    <p>
        Lorem ipsum dolor sit amet consectetur, adipisicing elit. Distinctio porro
            quos fugit eum corporis veritatis illo facilis, natus, reprehenderit 
            <a class="link" href="#"> Mas sobre este articulo </a> necessitatibus autem, enim doloribus
            corrupti fuga modi impedit aspernatur commodi dicta.
    </p>
  • CSS: 
    
    p{
        color: deeppink;
    }

    .link{
        color: inherit;
    }
  • El resultado que obtenemos es este:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Distinctio porro quos fugit eum corporis veritatis illo facilis, natus, reprehenderit Mas sobre este articulo necessitatibus autem, enim doloribus corrupti fuga modi impedit aspernatur commodi dicta.

El estilo de los p o parrafos, se heredo a este elemento href que por defecto no esta dispuesto a ser afectado por la herencia, ya que la propiedad inherit se puede aplicar para cualquier propiedad que por defecto no se herede pero queremos que se puede heredar.

Suprimir la herencia de un elemento:

Si queremos que un elemento que si esta siendo afectado por la herencia vuelva a su estado inicial es conveniente usar la propiedad initial la cual hace que el elemento al cual sea aplicada vuelva a su estado incial "de fabrica" ignorando cualquier tipo de herencia.

De esta forma:

  • HTML: 
    
    <h2> Esto es la herencia en <span class = "normal"> CSS </span> </h2>

    <span> CSS </span>
  • CSS: 
    
    h2{
        color: steelblue;
        font-size: 26px;
    }

    .normal{
        color: initial;
        font-size: initial;
    }
  • El resultado que obtenemos es este:
