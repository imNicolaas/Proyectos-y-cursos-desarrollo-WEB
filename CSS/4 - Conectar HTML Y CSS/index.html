Formas de conectar HTML y CSS

  1. Creando un archivo CSS en la carpeta del proyecto:

    Esta es la forma mas recomendada de crear una conexion entre HTML y CSS ya que de esta forma el contenido de los dos lenguajes esta separado y el proyecto goza de una mejor armonia y un mayor orden

    La conexion se haria primero creando el documento CSS que queremos vincular, en la misma carpeta del proyecto. Y en el documento HTML haremos uso de una etiqueta <link rel="stylesheet" href=""> que ira posicionada en el head del documento, donde el stylesheet hace referencia a "hoja de estilos" y en el " href " deberemos poner la direccion de la hoja de estilo CSS que queremos vincular, entonces automaticamente esa hoja de estilo quedara vinculada al documento HTML y a tu proyecto.

  2. Usando una etiqueta Style:

    Es posible hacer uso de el lenguaje CSS con una etiqueta HTML en el mismo documento sin hacer uso una hoja de estilo y es con la etiqueta <style> </style> pero aunque esto sea posoble no es recomendable en la gran mayoria de los casos ya que en un proyecto de desasrrollo de una pagina web se tienen varios documentos HTML y no es conveniente si se quiere cambiar algo en el estilo de la o las paginas

    esta etiqueta se veria de esta forma: 

    
                <style>

                    .titulo{
                        font-family: 'Oswald', sans-serif;
            
                        display: flex;
                        justify-content: center;
            
                        font-size: 40px;
                    }
            
                    body{
                        font-family: 'Roboto', sans-serif;
            
                        font-size: 17px;
                        
                    }   
                </style>

  4. Haciendo uso del @import

    Para implementar una hoja de estilo CSS en nuestro proyecto haciendo uso del @import solo se escribiria en el heead del socumento HTML y en url se pondria la direccion de la hoja de estilos 

    
    @import url(/HTML y CSS/CSS/Conectar HTML y CSS/CSS/styles.css)

    Esta es la menos recomendada de todas y se podria decir que su uso no es correcto ya que al implementar css de esta fomra en nuestra pagina, se carga todo de forma asincrona es decir que el HTML cargara todos los componentes de la pagina entre ellos este @import o sea que si tardan en cargar los demas elementos el estilo cargara despues haciendo que la pagina de una mala experiencia
