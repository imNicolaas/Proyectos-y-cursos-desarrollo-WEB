Mas atributos para formularios

Placeholder

Se representa por medio de un texto fantasma que da una pista a lo que el usuaroi tiene que introducir, de esta forma: 


<input type = "text" placeholder = "Introduce tu nombre">

Atributo required

Este atributo hace que el campo se vuelva obligatorio, de no ser rellenado el fomrulario no se podra enviar y se podra ver un mensaje que indica que campo se debe rellenar 


    <label for="correo">Correo electronico (este campo es obligatorio)</label>

    <input type="text"  id="correo" required>
Este campo es obligatorio

Atributo readonly

Este atributo hace que el inout sea solo de lectura, es decir que se podra seleccionar el texto en el pero no se podra cambiar el contenido del mismo pero lo que contenga este input si será enviado al formulario 


    <input type="text" value="Registro de docentes">

Atributo disabled

Este atributo a diferencia del readonly no se resalta al ser clickeado y esta informacion no será enviada al formulario cuando se le de enviar 


    <input type="text" value="input con atributo disabled" disabled>

Atributo min y max

Este atributo se usa para establecer el valor minimo y maximo de un input numerico si incumple con alguna de estas condiciones mostrara una alerta emergente, no es necesario que vayan los dos atributos, es decir que podemos hacer uso de el que deseemos. 


    <input type="number" min="10" max="100">

Atributo minlenght y maxlenght

Este atributo se usa para establecer el minimo y el maximo de caracteres que se pueden escribir en un input de texto 


    <input type="text" minlength="10" maxlength="15">

Atributo selected

Este atributo se usa para especificar cual sera la opcion que estara marcada por defecto en un select con varias opciones 


    <select name="numeros">

        <option value = "Uno"> Uno </option>
        <option value = "Dos" selected> Dos </option>
        <option value = "Tres"> Tres </option>

    </select>

Atributo autofocus

Este atributo se usa para dejar el cursor de texto en un campo en especifico, como si el usuraio ubuese hecho click en el campo para escribir pero es el campo que saldra por defecto 


    <label for = "campo_uno">A</label>
    <input type="text"  id="campo_uno" placeholder="campo uno">
    <br>

    <label for = "campo_dos">B</label>
    <input type="text"  id = "campo_dos" placeholder = "campo dos"  autofocus>
    <br>

    <label for = "campo_tres">C</label>
    <input type = "text"  id = "campo_tres" placeholder = "campo tres">
    <br>
