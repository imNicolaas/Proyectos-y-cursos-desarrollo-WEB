USO DE LA ETIQUETA AUDIO

El elemento audio se usa para insertar contenido de audio en un documento HTML o XHTML. El elemento audio se agregó como parte de HTML 5.

Esta etiqueta consta de los elementos src, para especificar la fuente de donde sale el audio en cuestión. autoplay, autobuffer, buffered, controls, loop, mozCurrentSampleOffset, que son
elementos de interacion con el audio.

Si se coloca solo la ruta para hacer uso de la etiqueta audio no se podra visualizar ningun tipo de contenido o reproductor 
    
    <audio src="/HTML y CSS/HTML//Contenido_Enbebido/assets/audio/Evgeny Grinko - Field.mp3"></audio>

Pero si se hace uso de el atributo controls si se podra ver, de esta forma:

    <audio src="/HTML y CSS/HTML/Contenido_Enbebido/assets/audio/Evgeny Grinko - Field.mp3" controls></audio>

Para que el contenidose empiece a reproducir de forma automatica se usa el atributo autoplay

Para que el audio aparezca de forma muteada se usa el atributo muted
