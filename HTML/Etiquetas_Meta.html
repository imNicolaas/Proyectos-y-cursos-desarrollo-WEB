Usos de la etiqueta meta

Definicion:

meta de "metainformation" - metainformación. Sirve para aportar información sobre el documento. 

    Sus etiquetas son: <meta> (solo tiene una).

    Está definido como: Elemento de cabecera.

    Crea una caja: No.

    Puede contener: Nada.

    Puede ser contenido por: Elementos head

La mayoria de las etiquetas meta solo utilizan dos atributos para su funcionamiento las cuales son: 

        name:  Hace referencia a que tipo de metadato le estamos dando al nevegador.

        content:  El contenido de ese metadato.

Atributos al completo:

        name = name [CS]: Este atributo identifica un nombre de propiedad. Esta especificación no enumera los valores legales para este atributo. 

        content = cdata [CS]: Este atributo especifica el valor de una propiedad. Esta especificación no enumera los valores legales para este atributo. 

        scheme = cdata [CS]: Este atributo especifica un esquema que se usará para interpretar el valor de la propiedad (véase la sección sobre perfiles para más detalles). 

        http-equiv = name [CI]: Este atributo puede utilizarse en lugar del atributo name. Los servidores HTTP utilizan este atributo para obtener información sobre los 
        encabezados del mensaje de respuesta HTTP.

Usos en las paginas comunes:

Viewport:

La estructura por defecto de una etiqueta meta de en una estructura basica HTML es: 

     <meta name="viewport" content="width=device-width, initial-scale=1.0">
la cual es de tipo viewport y solo funciona para hacer un diseño de pagina web adaptable a dispositivos moviles

Description:

Este etiqueta meta se usa para especificar el contenido que se va a previsualizar abajo del titulo de la pagina cuando esta este en la feed principal de el navegador

De esta forma:

su estructura es asi:

 <meta name="description" content="Aqui debe ir la descripcion que queremos que se vea como en el ejemplo anterior">

Author:

En esta etiqueta meta va el nombre del dueño de la web o quien la desarrollo, ya sea una empresa o un individuo

Su estructura es asi:

<meta name="Author" content="Nicolas Corredor Moya">
