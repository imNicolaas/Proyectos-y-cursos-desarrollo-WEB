Elementos extras para formularios

Etiqueta fieldset

Esta etiqueta (grupo de campos) permite organizar en grupos los campos de un formulario encerrandolos en una cuadricula a al que es posible ponerle un nombre con la etiqueta <legend></legend>

Datos personales

Etiqueta file

Control que permite al usuario seleccionar un archivo. Se puede usar el atributo accept para definir los tipos de archivo que el control podrá seleccionar.


Etiqueta meter

Esta etiqueta se usa para representar una barra de medicion la cual se le puede poner un valor minimo y uno maximo, y entre este intervalo se podran poner el valor por defecto que tendra al caragar la pagina y ademas con los atributos low se establecera lo que se considera como un valor bajo en el intervalo, con el atributo high se establece lo que se consifera un valor alto y con el atributo optimum se establecera lo que es un valor optimo.

Etiqueta progress

Se puede usar para representar el progreso de una tarea y tiene los atributos id, max y value.

Etiqueta textarea

Esta etiqueta representa un area de texto modificable en tamaño o con la potestad de ser bloqueada. Utiliza los atributos name, id, cols y rows
