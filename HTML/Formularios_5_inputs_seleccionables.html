Tipo de identificacion: (FORMULARIO SELECT)

Este tipo de formulario despliega las opciones que estan relacionadas con el tema de selcecion y todas van dentro de una etiqueta <select name=""> </select> y cada opcion va dentro de una etiqueta option con el atributo value que representa el valor que posee, asi:

            
                <select name="Tipo de identificacion">

                    <option value = "Cedula_de_ciudadania"> C.C </option>
                    <option value = "Tarjeta_De_identidad"> Tarjeta de identidad </option>
                    <option value = "Contraseña"> Contraseña </option>

                </select>

Para hacer que en este menu desplegable se puedan selecionar varias opciones se usa el atributo Multiple que al ser utilizado se podran selecionar varias opciones haciedno uso continuo de la tecla control y seleecionando las opciones deseadas.

Eliga su mascota: (Formulario seleccionables avanzados)

Este input es igual al de select pero se hace uso de la etiqueta <optgroup> </optgroup> para agrupar diferentes opciones en grupos y se les pone nombre con la etiqueta label, para hacer mas facil el proceso de seleccion, de esta forma: 

        
            <select name="Mascotas">

                <optgroup label="4 patas">
                    <option value="Perro">Perro
                    <option value="Gato">Gato
                    <option value="Hamster">Hamster
                    <option value="Conejo">Conejo
                </optgroup>
                
                <optgroup label="Aves">
                    <option value="Loro">Loro
                    <option value="Canario">Canario
                </optgroup>
                
                <optgroup label="Otros">
                    <option value="Serpiente">Serpiente
                    <option value="T_rex">T rex
                    <option value="Dodo">Dodo
                </optgroup>
                
            </select>

Seleccion de mascotas: (FORMULARIOS TIPO DATALIST)

Muestra las opciones que estén dentro de esta etiqueta en forma de lista dinámica es decir que en el input principal se podrá digitar una parte de la opción que estemos buscando entre la lista y las opciones que coincidan deberán salir recomendadas
