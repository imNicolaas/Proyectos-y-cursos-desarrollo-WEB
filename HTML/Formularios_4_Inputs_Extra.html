Inputs extra

Input tipo color:

Sirve para seleccionar un color en caso de ser necesitado


Input tipo numero:

Solo deja digitar numeros y posee widgets para avanzar en los numeros digitados


Input tipo range:

Se utiliza para seleccionar un lugar especifico en una barra de rango, ademas de que emplea el atributo step el cual determina el numero de posiciones que avanzara la barra, el atributo min que determinara el numero minimo de la barra de rango, el atributo max que determinara el valor maximo de la barra de rango


Input tipo reset:

Se utiliza generalmente para resetear el formulario y dejarlo con los valores por defecto


Input tipo text:

Es el equivalente a un input por defecto pero se le pueden dar mas funciones por medio de JavaScript
