Atributos de Accesibilidad

los atributos de Accesibilidad se usan para darle o no permisos a las personas de acceder al contenido de la pagina, tambien se utiliza com ayuda para gente invidente

• Estos son algunos de los atributos de accesibilidad:

  1. tabindex: Este atributo de usa para indicar el orden en el que se va desplazar el usuario entre las secciones de la pagina usando la tecla tabulador por ejemplo si usamos tres parrafos de ejemplo de esta forma 
    
                <p tabindex="1">Parrafo 1: Lorem ipsum dolor sit amet consectetur 
                       adipisicing elit. Dolores laudantium dolorem sed neque cum reiciendis
                       totam? Id maiores sapiente laboriosam facilis voluptate, dolor, deleniti
                       ipsa quisquam aliquam dicta, consequatur dolores. </p>

                <p tabindex="3">Parrafo 1: Lorem ipsum dolor sit amet consectetur 
                       adipisicing elit. Dolores laudantium dolorem sed neque cum reiciendis
                       totam? Id maiores sapiente laboriosam facilis voluptate, dolor, deleniti
                       ipsa quisquam aliquam dicta, consequatur dolores. </p>

                <p tabindex="2">Parrafo 1: Lorem ipsum dolor sit amet consectetur

    2. • Atributos para gente invidente:

  2. aria-label:Las personas invidentes usan extensiones creadas para el navegador con el proposito de interpretar la web y explicar por comandos de voz en que parte de la web esta el usuario invidente, en el caso de google chrome es la extension Chorme Vox Classic y lo que hace es leer el contenido que tengamos en la web para que las personas invidentes lo escuchen.

    y con el atributo aria-label podemos escribir una descripcion de lo que estamos representando en esa seccion de la pagina web asi sin afectar el contenido de la misma, se le sera explicada el lector invidente de esta forma:

    Parrafo 1: Bitcoin cae desmesuradamente... Lorem, ipsum dolor sit amet consectetur adipisicing elit. Debitis reprehenderit itaque aliquam iure quaerat voluptatem vero, dolor perspiciatis quae dolorum in odit ipsum magni incidunt laudantium modi deserunt, porro autem.

  3. Role: le da informacion al navegador sobre el elemento sobre el cual nos estamos posicionando, por ejemplo si se esta hablando sobre un link de un articulo se colo caria el rol de link 
    <a href="articulo" role="link"
