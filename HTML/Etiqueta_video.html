USO DE LA ETIQUETA VIDEO

• Etiqueta video: 
    
    El elemento video se utiliza para incrustar vídeos en un documento HTML o XHTML Esta etiqueta consta de los elementos src, para especificar la fuente de donde sale el video en 
    cuestión. autoplay, autobuffer, buffered, controls, height, loop, poster width que son elementos de interacion con el video.

• Para hacer que un video se ajuste siempre al ancho minimo de una pagina se usan las siguintes propuiedades de la etiqueta estilo 

        video{
            max-width: 100%;
        }

• Para hacer que un video se reproduzca solo se usa el atributo autoplay

• Para hacer que un vide este muteado por defecto se usa el atributo muted

• Para hacer que un video se reproduzca sin parar se usa el atributo loop

• Para definir una imagen de previsualizacion o una "miniatura" se usa el atributo poster, si este no se especifica la imagen de previzualizacion.
