Tipos de inputs

inputs tipo date

• input tipo date basico

• input tipo datetime (no compatible con firefox):

Se puede usar para poner la fecha(dia/mes/año) y ademas la hora

• input tipo date y uno tipo time juntos para reemplzar el de tipo datetime-local

• input tipo mes:

ademas de especificar un mes en espacifico es posible especificar el año en el que se ubica el mes del que se esta hablando

• input tipo week:

Con este input se puede especificar una semana espacifica de un año
