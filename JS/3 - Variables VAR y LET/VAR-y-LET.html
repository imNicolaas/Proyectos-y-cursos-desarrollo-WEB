Variables VAR y LET en Javascript

¿Que es una variable?

Es un lugar en la memoria de nuestro dispositivo y un nombre asociada a dicho espacio, usada para guardar datos o informacion.

Los identificadores deben comenzar con:

  • Una letra
  • Un signo de dolar $
  • Un guión bajo
  • Nunca con números o caracteres especiales

Ambito de existencia o scope de las Variables

  • Scope global y scope funcional:

Antes en javascript no habia ambito de bloque es decir que una variable que se creara dentro de un bloque de codigo (segmento de codigo separado por corchetes "{ }" ) como una estructura repetitiva, etc, no existia solo dentro de ese bloque sino que el navegador la declaraba al principio del documento haciendo que existiera globalmente "en el ambito global o scope global", y ademas si la variable se declaraba dentro de una funcion esta era declarada al principio de la funcion "scope funcional".

  • Scope o ambito de bloque:

Pero ahora se cuenta con este "ambito de bloque" en javascript pero para hacer uso de este se tiene que declarar la varible con la sentencia o palabra reservada let en vez de var, es decir que si se declara una variable con la palabra var esta va a tener el scope global y funcioal.

mientas que si se declara con la palabra let esta podra hacer usod e el scope de bloque

Objeto Window de javascript:

Es el objeto principal en la jerarquía y contiene las propiedades y métodos para controlar la ventana del navegador. De él dependen todos los demás objetos de la jerarquía. Vamos a ver la lista de sus propiedades y métodos.

El objeto window representa la ventana que contiene un documento DOM; la propiedad document apunta al DOM document cargado en esa ventana. El objeto window al que pertenece un documento puede ser obtenido usando la propiedad document.defaultView.

Objeto Window y variables let y var

Todas las variables que se declaren con var es decir en un ambito global van a estar dentro de este objeto window es decir que se podran buscar haciendo un console.log(window).

mientras que las que se delcaren con let no estaran disponibles dentro de este objeto "Window"

Ejemplo:

  • En JS:

        let a = 2
    var b = 3

    console.log(window.a)
    console.log(window.b)

  • Resultado en consola:

        undefined
    3
