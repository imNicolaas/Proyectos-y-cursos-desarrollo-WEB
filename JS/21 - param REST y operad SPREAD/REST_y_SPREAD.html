REST y SPREAD en JS

Parametro REST:

El parametro REST es una forma de hacer que a una funcion se le puedan pasar parametros infinitos o los parametros que se quieran pasar en una ejecución de una función sin que estos parametros ya esten definidos para que la funcion las reciba, si no que con la sintaxis REST se pueden pasar y el metodo los pueda usar dinamicamente.

Su sintaxis es poner tres puntos (...) al inicio de el parametro que se quiera definir y entender como el parametro REST y al que se la va a poder dar dinamismo con los demas valores que se pueden insertar infinitamente.

        function sumar (a, b, ...c){
            resultado = a+b;
        
                c.forEach(function(el){
                    resultado += el
                });
        
            return resultado;
        }
        
        console.log(sumar(2,2,   1,1,1))

Operador SPREAD (Operador de Propagación):

Es un operador el cual hace que todos los elementos de un arreglo u objeto puedan ser usados para formar parte de nuevos arreglo/objeto junto con otros elementos en su interior (es decir hace que se pueden sumar los elementos de un arreglo/objeto existente en otro), o para ser asignados a variables, constantes o funciones.

Ejemplo:

  • En JS:

    
//Spread operator: Cuando se tenga una expresion con parametros establecidos y se quiera crear otro en el que
se quieran agregar todos los valoes de alguno que ya ser tenga creado, el SPREAD se puede usar en estos casos

        const arr1 = [1,2,3,4,5], arr2 = ['a', 'b', 'c', 'd', 'e']
        console.log(arr1, arr2);//se tienen dos arreglos de 5 posiciones 

//si se quiere crear un arreglo que sea la mezcla de los dos que se tienen se podria intentar.

        const arr3 = [arr1, arr2];//pero asi se tiene un arrelgo de dos posiciones
        y con dos arreglos de 5 pos cada una.

        console.log(arr3)

//pero con el Operados SPREAD o de propagacion se pueden juntar los dos arreglos con sus elementos

        const arr4 = [...arr1, ...arr2];
        console.log(arr4)

        const asd = [1,2,3, ...arr2]

        console.log(asd)

//Operador SPREAD en objetos

const a = {
    a: 'a',
