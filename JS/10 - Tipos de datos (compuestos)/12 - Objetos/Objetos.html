Objetos en JS

En JavaScript, un objeto es un entidad independiente con propiedades y tipos, como una, por ejemplo. Una taza es un objeto con propiedades. Una taza tiene un color, un diseño, un peso.

Su sintaxis es el constructor const, let o var seguido del nombre, igual y despues las Curly Braces, dentro de los corchetes van las propiedades con sus valores, que son como variables que se adjuntan al objeto:

Dentro de un objeto a las variables se le van a llamar atributos/propiedades/keys y a las funciones se les llama métodos


    let objetoA = {
        propiedad1: 'valor1',
    }

Una propiedad puede tener como valor cualquier tipo de dato, incluso funciones.

Si una propiedad no tiene valor se le asigna undefined.

Si se pueden usar const para definir un objeto por que para js el valor de el objeto es su nombre y mientras no cambie, este seguira valiendo lo mismo, asi el objeto en su interior si su nombre o apuntador no cambia para js seguira siendo constante.
