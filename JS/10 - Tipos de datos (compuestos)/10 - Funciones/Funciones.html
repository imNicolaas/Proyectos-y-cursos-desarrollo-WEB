Funciones en JS

Las funciones son bloques de codigo autocontenido, es decir que...

Al que despues de haberlo escrito uno vez se le va a poder llamar las veces que queramos y se le podran pasar los valores que necesitemos en cada vez que se llame.

Puede recibir parametros y devolver valores.

En js Las funciones pueden retornar valores, pero ademas se puede hacer que retornen como valor la ejecucino de otra funcion

Ademas una variable puede tomar valor de una funcion.

Tipos de funciones

Existen dos tipos de funciones o formas de declararlas:

  • Funciones declaradas: Para este tipo de delcaracion se usa la palabra reservada function y su sintaxis es function nombre_de_la_funcion (parentesis) {cuerpo de la funcion}
  • Funciones expresadas.

Invocacion de funciones:

Para invocar una funcion se debe poner el nombre de la funcion mas los parentesis

  • En JS:

    
function funcion_1 (){
    console.log("1");
    console.log("2");
    console.log("3");
}

funcion_1();
