Condicionales en JS

Depende de el evaluar alguna condicion

El principal de todos es el If Else, y para anidar mas condiciones se puede usar un if-else if-else

Ejemplo:

  • En JS:

    
    if(edad >=18){
        console.log('mayor de edad')
    }else{
        console.log('menor de edad')
    }

Operador ternario:

Para simplificar el if else

Sintaxis:

La sintaxis del operador ternario consta de la condicion que se va a evaluar encerrada entres parentesis, seguido por un signo de interrogacion y la lo que se va a ejecutar si la condicion es verdadera, seguido de : para la con lo que ejecutara si la condicion es falsa. 

        
    (condicion) ? 'evento de verdadero': 'evento de falso';

Ejemplo:

    let eresMayor = (edad>=18) ? 'mayor de edad': 'menor de edad'; 

    console.log(eresMayor);

switch - case

Se usa cuando se tengan diferentes valores para una misma variable.

Sintaxis:

    switch (key (llave o variable a evaluar) ) {
    case value (caso en el que la variable tenga este valor):
        (Instruciones que se ejecuran en este caso)
    break; (Punto de break, dejara de evaluar este case se haya cumplido o no)

    default: (Condicion por defecto, si ningun caso cubre el valore de la varialbe)
    (Instruciones que se ejecuran en este caso)
    break; (Punto de break)
    }

Ejemplo:

    let dia = 2;

    switch (dia) {
        case 0:
            console.log('Es domingo')    
        break;
    
        case 1:
            console.log('Es lunes')    
        break;
    
        case 2:
            console.log('Es miercoles')    
        break;
    
        case 3:
            console.log('Es jueves')    
        break;
    
        case 4:
            console.log('Es viernes')    
        break;
    
        case 6:
            console.log('Es sabado')    
        break;
    
        default: //si la variable en cuestion no cae en ningun caso
            console.log('No es un valor valido para dia')
        break;
    }
