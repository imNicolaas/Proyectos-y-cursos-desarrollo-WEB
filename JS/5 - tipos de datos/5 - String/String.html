Strings o "Cadenas de texto" en JS

Es una secuencia ordenada de elementos o caracteres de cualquier tipo.

Se puede indicar una cadena de texto con comillas dosbles (" ") o comillas simples (' '').

Ejemplo:

  • En JS:

    let texto = 'Hola mundo';

Pero ademas existe una forma mas formal de declar un String y tiene la siguiente sintxis

  • En JS:

    let texto = new String('Hola mundo');

const texto_2 = new String('¡Hola mundo!');

Los Strings en javascript tiene tanto propiedades como metodos, siendo las propiedades formas de decribir al String existente y sus carecteristicas, y los metodos acciones que se peuden tomar en base a este dato "String", además los metodos terminan con ().
